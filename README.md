# simplic-service
Contains a set of simplic libraries (services) which can be used standalone or within the simplic framework. The services use dependency injection (unity container).

# How to contribute
First read our [conventions](https://simplic.github.io/dev/csharp/getting_started/conventions.html).

- create a service core project: you just create interfaces and pocos. no implementation here
- implement the service interface
- create unit tests
- create a pull request and ask [Benedikt Eggers](https://github.com/simplicbe) and [Yavuz Günay](https://github.com/yvz5) to review it.
